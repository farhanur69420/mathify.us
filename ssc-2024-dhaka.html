1. If \(a+b=5\) and \(ab=6\), then \(a^2+b^2=\)?

▼

Bengali: যদি \(a+b=5\) এবং \(ab=6\) হয়, তাহলে \(a^2+b^2\) = ?

Options:

  1. 11
  2. 12
  3. 13
  4. 25

Correct Answer: C

Solution

Use the identity: \(a^2+b^2 = (a+b)^2 - 2ab\)

(5)^2 - 2(6) = 25 - 12 = \boxed{13}
Note: 🔑 This identity appears in almost every SSC paper. Memorize it as the '2ab trick'.

2. Which is an irrational number?

▼

Bengali: নিচের কোনটি অমূলদ সংখ্যা?

Options:

  1. \(\frac{22}{7}\)
  2. \(\sqrt{4}\)
  3. \(\sqrt{3}\)
  4. \(0.\overline{3}\)

Correct Answer: C

Solution

\(\frac{22}{7}\) — rational (ratio of two integers)

\(\sqrt{4}=2\) — integer, hence rational

\(0.\overline{3}=\frac{1}{3}\) — repeating decimal, rational

\(\sqrt{3}=1.7320508...\) — non-terminating, non-repeating → irrational

Note: 🔑 \(\sqrt{n}\) is irrational when \(n\) is not a perfect square

3. \(\log_2 32 = ?\)

▼

Bengali: \(\log_2 32\) এর মান কত?

Options:

  1. 4
  2. 5
  3. 6
  4. 16

Correct Answer: B

Solution

Ask: 2 to what power gives 32?

2^1=2,\ 2^2=4,\ 2^3=8,\ 2^4=16,\ 2^5=32
\log_2 32 = \boxed{5}
Note: 🔑 \(\log_b x = n \iff b^n = x\)

4. If \(p=\sqrt{5}+\sqrt{3}\) and \(q=\sqrt{5}-\sqrt{3}\), then \(pq=\)?

▼

Bengali:

Options:

  1. \(\sqrt{2}\)
  2. \(2\)
  3. \(8\)
  4. \(2\sqrt{15}\)

Correct Answer: B

Solution

Difference of squares: \((a+b)(a-b)=a^2-b^2\)

pq = (\sqrt{5})^2-(\sqrt{3})^2 = 5-3 = \boxed{2}
Note: 🔑 Conjugate surds always give a rational product.

5. \(\dfrac{\sqrt{72}}{\sqrt{8}}\) simplifies to:

▼

Bengali: \(\dfrac{\sqrt{72}}{\sqrt{8}}\) এর সরলতম মান:

Options:

  1. \(\sqrt{6}\)
  2. \(2\sqrt{2}\)
  3. \(3\)
  4. \(9\)

Correct Answer: C

Solution
\sqrt{\frac{72}{8}} = \sqrt{9} = \boxed{3}
Note: Rule: \(\frac{\sqrt{a}}{\sqrt{b}}=\sqrt{\frac{a}{b}}\)

6. If \(\log x = \frac{1}{2}\), then \(x = ?\)

▼

Bengali: \(\log x = \frac{1}{2}\) হলে \(x\) = ?

Options:

  1. \(2\)
  2. \(5\)
  3. \(\sqrt{10}\)
  4. \(100\)

Correct Answer: C

Solution

Unwritten base means \(\log_{10}\)

x = 10^{1/2} = \boxed{\sqrt{10}}

7. \(n(A)=25,\ n(B)=20,\ n(A\cap B)=10\). Find \(n(A\cup B)\).

▼

Bengali:

Options:

  1. 30
  2. 35
  3. 45
  4. 55

Correct Answer: B

Solution

Inclusion-exclusion principle:

n(A\cup B) = n(A)+n(B)-n(A\cap B) = 25+20-10 = \boxed{35}

8. If \(A=\{1,2,3\}\), number of subsets = ?

▼

Bengali: \(A=\{1,2,3\}\) এর উপসেটের সংখ্যা কত?

Options:

  1. 4
  2. 6
  3. 8
  4. 9

Correct Answer: C

Solution
2^{|A|} = 2^3 = \boxed{8}
Note: Subsets: ∅, {1}, {2}, {3}, {1,2}, {1,3}, {2,3}, {1,2,3}

9. \(f(x)=2x+3\). Find \(f(f(1))\).

▼

Bengali: \(f(x)=2x+3\) হলে \(f(f(1))\) = ?

Options:

  1. 11
  2. 13
  3. 15
  4. 17

Correct Answer: B

Solution
f(1) = 2(1)+3 = 5
f(f(1)) = f(5) = 2(5)+3 = \boxed{13}

10. Which set is a subset of every set?

▼

Bengali: কোন সেটটি প্রতিটি সেটের উপসেট?

Options:

  1. Universal set
  2. Power set
  3. Empty set ∅
  4. Singleton set

Correct Answer: C

Solution

The empty set ∅ has no elements, so vacuously it satisfies the subset condition for every set.

\emptyset \subseteq A \quad \text{for every set } A

11. \(\log 2 + \log 5 = ?\)

▼

Bengali: \(\log 2 + \log 5\) এর মান কত?

Options:

  1. 1
  2. \(\log 3\)
  3. \(\log 7\)
  4. 10

Correct Answer: A

Solution
\log 2 + \log 5 = \log(2\times5) = \log 10 = \boxed{1}
Note: 🔑 Product rule: \(\log a + \log b = \log(ab)\). And \(\log_{10}10=1\) always.

12. \(a^2-b^2=21,\ a-b=3\). Find \(a+b\).

▼

Bengali: \(a^2-b^2=21,\ a-b=3\) হলে \(a+b=\)?

Options:

  1. 6
  2. 7
  3. 8
  4. 9

Correct Answer: B

Solution
a^2-b^2=(a+b)(a-b)
21 = (a+b)\times3 \implies a+b = \boxed{7}

13. Rationalize \(\dfrac{1}{\sqrt{3}-1}\):

▼

Bengali: \(\dfrac{1}{\sqrt{3}-1}\) এর হর মূলদ করলে পাওয়া যায়:

Options:

  1. \(\frac{\sqrt{3}-1}{2}\)
  2. \(\frac{\sqrt{3}+1}{2}\)
  3. \(\sqrt{3}+1\)
  4. \(\sqrt{3}-1\)

Correct Answer: B

Solution

Multiply by conjugate \((\sqrt{3}+1)\):

\frac{1}{\sqrt{3}-1}\cdot\frac{\sqrt{3}+1}{\sqrt{3}+1} = \frac{\sqrt{3}+1}{3-1} = \boxed{\frac{\sqrt{3}+1}{2}}

14. \(\log_a a^3 = ?\)

▼

Bengali: \(\log_a a^3\) এর মান কত?

Options:

  1. 1
  2. 2
  3. 3
  4. \(a\)

Correct Answer: C

Solution
\log_a a^3 = 3\log_a a = 3\times1 = \boxed{3}
Note: Power rule: \(\log_b x^n = n\log_b x\)

15. Sum of angles in a triangle:

▼

Bengali: একটি ত্রিভুজের কোণের সমষ্টি:

Options:

  1. 90°
  2. 120°
  3. 180°
  4. 360°

Correct Answer: C

Solution
\angle A + \angle B + \angle C = \boxed{180^\circ}
Note: Proved by drawing a line parallel to base through the apex — alternate angles fill out a straight angle (180°).

16. In △ABC, AB=AC, ∠B=70°. Find ∠A.

▼

Bengali: △ABC-এ AB=AC, ∠B=70° হলে ∠A = ?

Options:

  1. 30°
  2. 40°
  3. 50°
  4. 70°

Correct Answer: B

Solution

Isosceles → base angles equal → ∠B = ∠C = 70°

\angle A = 180^\circ - 70^\circ - 70^\circ = \boxed{40^\circ}

17. An exterior angle of a triangle equals:

▼

Bengali: একটি ত্রিভুজের বহিঃস্থ কোণ সমান:

Options:

  1. One interior angle
  2. Sum of all interior angles
  3. Sum of two non-adjacent interior angles
  4. Half the sum

Correct Answer: C

Solution
\text{Exterior angle} = \alpha + \beta \quad (\text{two non-adjacent interior angles})
Note: Exterior Angle Theorem — follows directly from angles on a straight line summing to 180°.

18. The angle in a semicircle is:

▼

Bengali: অর্ধবৃত্তস্থ কোণের মান:

Options:

  1. 45°
  2. 60°
  3. 90°
  4. 120°

Correct Answer: C

Solution

Thales' theorem: angle subtended by a diameter at any point on the circle is always 90°.

\angle = \boxed{90^\circ}

19. Distance from (3, 4) to the origin:

▼

Bengali: (3,4) বিন্দু থেকে মূলবিন্দুর দূরত্ব:

Options:

  1. 3
  2. 4
  3. 5
  4. 7

Correct Answer: C

Solution
d = \sqrt{3^2+4^2} = \sqrt{9+16} = \sqrt{25} = \boxed{5}
Note: Classic 3-4-5 right triangle!

20. Slope of a line parallel to the x-axis:

▼

Bengali: x-অক্ষের সমান্তরাল রেখার ঢাল:

Options:

  1. Undefined
  2. 1
  3. 0
  4. −1

Correct Answer: C

Solution

A horizontal line has zero vertical change (Δy = 0).

m = \frac{\Delta y}{\Delta x} = \frac{0}{\Delta x} = \boxed{0}

21. If \(\tan\theta=\frac{3}{4}\), then \(\sin\theta=\)?

▼

Bengali: \(\tan\theta=\frac{3}{4}\) হলে \(\sin\theta=\)?

Options:

  1. \(\frac{4}{5}\)
  2. \(\frac{3}{5}\)
  3. \(\frac{4}{3}\)
  4. \(\frac{5}{3}\)

Correct Answer: B

Solution

opp = 3, adj = 4 → hyp = √(9+16) = 5

\sin\theta = \frac{\text{opp}}{\text{hyp}} = \boxed{\frac{3}{5}}

22. \(\sin^2\theta + \cos^2\theta = ?\)

▼

Bengali: \(\sin^2\theta + \cos^2\theta\) এর মান:

Options:

  1. 1
  2. 0
  3. \(\sin\theta\)
  4. 2

Correct Answer: A

Solution

From unit circle: point (cosθ, sinθ) lies on \(x^2+y^2=1\)

\sin^2\theta + \cos^2\theta = \boxed{1} \quad \forall\theta
Note: The fundamental Pythagorean identity — it is never anything other than 1.

23. \(\cos 60° = ?\)

▼

Bengali:

Options:

  1. \(\frac{1}{2}\)
  2. \(\frac{\sqrt{3}}{2}\)
  3. \(\sqrt{3}\)
  4. 1

Correct Answer: A

Solution

30-60-90 triangle: side ratios 1 : √3 : 2

\cos 60^\circ = \frac{\text{adj}}{\text{hyp}} = \frac{1}{2}

25. Median of 3, 7, 4, 9, 2, 6, 1:

▼

Bengali: 3, 7, 4, 9, 2, 6, 1 এর মধ্যমা কত?

Options:

  1. 3
  2. 4
  3. 6
  4. 7

Correct Answer: B

Solution

Sort: 1, 2, 3, 4, 6, 7, 9 → n=7, middle = 4th value

\text{Median position} = \frac{n+1}{2} = 4\text{th} = \boxed{4}

28. Area of circle with r = 7 cm (π = 22/7):

30. HCF of 12 and 18:

▼

Bengali: 12 এবং 18 এর গ.সা.গু কত?

Options:

  1. 3
  2. 4
  3. 6
  4. 9

Correct Answer: C

Solution
12 = 2^2\times3,\quad 18 = 2\times3^2
\text{HCF} = 2^1\times3^1 = \boxed{6}
